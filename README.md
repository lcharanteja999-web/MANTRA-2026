# Kingdom Roads

A simple static travel website built for the MANTRA 2026 Summer School Frontend Assignment.

## Pages
- `index.html` – Home
- `about.html` – About
- `destinations.html` – Destinations
- `packages.html` – Packages with cost calculator
- `gallery.html` – Photo gallery with category filter
- `booking.html` – Booking/Enquiry form
- `contact.html` – Contact details

## Technologies Used
- HTML5
- CSS3
- JavaScript

## Features
- Responsive design for mobile, tablet, and desktop
- Basic SEO optimization
- Package cost calculator
- Booking form validation
- Gallery category filter
- Tawk.to live chat integration
